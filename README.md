- 初始化
````
composer install

````
- 拷贝配置文件
````

cp .env.example .env



````
- 生成Key
````
php artisan key:generate

````
- 导入数据库文件
新建数据库tms,在public/sql目录的sql文件导入数据库


- 启动服务
````
php artisan server

````
